# Add A Docc Catalog

Use this page when the package already has symbol docs and now needs a `.docc`
catalog, landing page, articles, or topic-group curation.

## Fast Path

1. Confirm the package/catalog layout with [Documenting a Swift framework or package](../assets/DocCDocumentation.docc/documenting-a-swift-framework-or-package.md).
2. Add articles or extension files with [Adding supplemental content to a documentation catalog](../assets/DocCDocumentation.docc/adding-supplemental-content-to-a-documentation-catalog.md).
3. Shape the landing page and topic groups with [Adding structure to your documentation pages](../assets/DocCDocumentation.docc/adding-structure-to-your-documentation-pages.md).
4. If you need `append` or `override`, check [DocumentationExtension](../assets/DocCDocumentation.docc/Reference%20Syntax/API%20Reference%20Syntax/DocumentationExtension.md).
5. If the catalog includes rich content, continue with [formatting-and-assets.md](formatting-and-assets.md) or [tutorial-workflow.md](tutorial-workflow.md).
