# GitHub publish command summary

Use this as the authoritative publish-domain `ghops` command map referenced by
`skills/github/SKILL.md`.

## Fast picks

- Current-branch context: `scripts/ghops publish context`
- Current-branch PR open or reuse: `scripts/ghops publish open`
- Explicit PR creation: `scripts/ghops publish create`
- Lifecycle changes: `scripts/ghops publish draft`,
  `scripts/ghops publish ready`, `scripts/ghops publish merge`,
  `scripts/ghops publish close`, `scripts/ghops publish reopen`
- Local checkout side effect: `scripts/ghops publish checkout`

- `scripts/ghops publish context`
