# GitHub publish command summary

Use this as the authoritative publish-domain `ghops` command map referenced by
the bundled `github` skill.

## Fast picks

- Current-branch context: `ghops publish context`
- Current-branch PR open or reuse: `ghops publish open`
- Explicit PR creation: `ghops publish create`
- Lifecycle changes: `ghops publish draft`,
  `ghops publish ready`, `ghops publish merge`,
  `ghops publish close`, `ghops publish reopen`
- Local checkout side effect: `ghops publish checkout`

## Commands

- `ghops publish context`
- `ghops publish open`
- `ghops publish create`
- `ghops publish draft`
- `ghops publish ready`
- `ghops publish merge`
- `ghops publish close`
- `ghops publish reopen`
- `ghops publish checkout`
